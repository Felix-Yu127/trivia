通常的全域設定：
為獲得瀏覽器良好支援，須使用瀏覽器前娺
 -moz-：適用於 firefox 瀏覽器。
 -webkit-：適用於 safari, google chrome,opera 等瀏覽器。
 -o-：適用於 opera 瀏覽器。
 -ms-：適用於 ie 瀏覽器

常用CSS的全域設定：

@import：導入，利用@import "about.less";指令可以將"about.less"裡的檔案內容導入到另一個檔案裡面

「*」：代表所有元素

「>」：底下所有符合的class都做選擇

「+」：

「&」：在同一層關係

「-」：

「:first-child」：第一個孩子

「:last-child」：最後一個孩子

display:block - 區塊，元素會以區塊方式呈現，除非設定 position 或 float，block 的特性是可以設定高度（height）、寬度（width）、上方與下方距離，像是 div、p、ul、li 均屬 block

display:inline - 所有文字或圖片均不換行，也就是全部都會是同一行的意思，inline 高度與寬度都不能設定，文字或圖片所佔的寬度就是他的寬度，像是 span、a、input、img、em 均屬 inline

margin collapsing：崩塌，兩個都是display:block的時候，margin之間都沒有任何障礙物，兩個margin就會重疊。
 (1)鄰近的兄弟姊妹之間 (2)當爸爸跟第一個孩子or最後一個孩子之間 (3)空的block本身會崩塌

box-sizing：盒子大小，其中屬性border-box，實際寬度是把border加上padding加上內容，由電腦自動計算，這樣才不會超出設定的寬度

font-family：字體系列(設定在css中)，從第一個字型開始，如果使用者無此字體型式就用第二個字型以此類推，如果都沒有會自動用使用者內建的

Source Sans Pro：字體的一種

sans-serif：無襯線字體(字體的一種)

font-weight：字體重量(字體粗細)，100最細，900最粗，400為中等

margin：邊界，邊界有時候可以重複，是跟其他物體之間的距離，比較像是，書跟其他物品之間的距離像是跟隔壁的筆差了 20cm、跟杯子隔了 30cm

padding：留白，距離邊界有多遠，是對本身的留白, 有點像是書裡面的留白，文字跟書之間的距離

transparent：透明

slider：滑塊

category：類別

recent-product：近期產品

laptop：筆記本電腦

summary：摘要

rounded：圓

circle：圈

description：描述，簡寫desc可以拿來當class的標題用

dot：點

intro：介紹

product：產品

recommended：推薦的

promo-info：宣傳信息

progress-bar：進度條

progress：進度

team-member：團隊成員

graphic design：平面設計

count：計數

git

common-list：常見的列表

autobiography：自傳

a[href=#][target=_bland]{view map}：縮寫的方式範例，展開後是<a href="#" target="_bland">view map</a>

copyright：版權

result：結果

text-align：文本對齊，文字對齊方式。是針對文章內的文字

center：置中

display：顯示

inline：直列，inline包不住block，就像紙包不住火一般。

block：區塊

容器置中的方式：將容器設定成區塊，設定寬度，設定margin上下0左右auto

img在預設是display:inline，系統會在底下預留一點點空白(文字底線下的空間)，所以容器的高度會不一樣，設成block就可以解決

Navigation：導航(導覽)

both：都，屬於clearfix中clear的屬性

list-style-type：列表樣式類型，屬性選none可以讓li的點點不見

none：沒有

text-decoration：文字修飾，可以讓a連結的樣式改變，如清掉底線

text-transform：文字轉換，其中屬性有lowercase(小寫)、uppercase(大寫)、initial(初始)、inherit(繼承)、capitalize(字首大寫)、none

transform：轉變，css3的功能，讓網頁元素呈現出旋轉（rotate）、縮放（scale）、移動（move）以及傾斜（skew）的特殊效果

transform: rotate()：旋轉，rotate(x,y,z)，根據設定的角度，以參考點為中心軸進行 2D、3D 旋轉。單位是(deg)

transform: translate()：翻，translate(x,y)，由參考點進行 2D 轉換特效，向 x 軸移動 x 距離，Y 軸移動 y 距離。單位是(%或px)

transform: scale()：縮放，scale(x,y)，由參考點進行 2D 縮放特效，X 軸方向縮放 x 倍，Y 軸方向縮放 y 倍。沒有單位，要輸入原比例跟改變比例(1,1.5)

transform: skew：傾斜，skew(x,y) 以參考點為中心，沿著 X 軸方向傾斜 x 度，沿著 Y 軸方向傾斜 y 度。單位是(deg)

border：邊界(邊框)，可加上right或bottom(上下左右都行)設定邊框的粗細、型式、顏色

solid：實線

dotted：虛線

hidden：隱藏線

border-radius：邊界半徑，，倒圓的效果，設定寬度的一半就能變成圓型

::before：前面，偽元素，用兩個::開頭，在目標物的前面加上指定的東西

::after：後面，偽元素，用兩個::開頭，在目標物的後面加上指定的東西，例如.boundary::after { content: "|";}，在.boundary後面加上一個「|」符號

content：內容

container：容器

line-height：行高(預設是倍數)，可直接設參數1、2字元高(1、2個字元高)或px(100個px高)，讓高度跟旁邊的圖一樣高就能讓裡面的文字高度置中，如logo高100px，line-height就設定100px

position：位置

relative：相對的

absolute：絕對

letter-spacing：字母間距，要注意單位px

hr：分隔線，hr不是實心線，是用border去湊出來的，所以可以設定高度，背景顏色，並把border設none，就能得到純高度的hr

background:背景

background-size：背景大小，其中屬性cover，讓背景左右符合顯示範圍

background-position：背景位置，左右、上下各50%就能圖片置中

background-attachment：背景附件，可選fixed、scroll、inherit、initial、local、unset

fixed：固定

scroll：滾動

inherit：繼承

initial：初始

local：本地

unset：未設置

rgba：background屬性，代表Red(紅色)Green(綠色)Blue(藍色)和Alpha的色彩空間，編寫方式 background: rgba(255,255,255,0.9)，就是白色透明度90%

Alpha：一般用作不透明度的參數，0就是完全透明，1就是完全不透明，0.5就是半透明

opacity：不透明度，0為完全透明，1為完全不透明

transition：漸變(過渡)，CSS3的功能，先選秒數，再選變化型態，再選對象all、none，常用變化型態有linear(線型)、ease(緩解)

iframe：從設定的網址裡面把網頁的內容塞進這個標籤(type)裡面，可以應用在貼上google map上

Table：表格

table：表格，table(表格) 裡面有 row(列) 裡面有 cell(元件)

tr：table row，表格中的一列一列(橫列)

td：table data，在每一列中的資料，跟每一欄是一樣的意思(直欄)

table結構可區分為thead(頭)、tbody(身體)、tfoot(尾)

th：在thead裡面的td，使用th來取代，其實跟td是一樣的，只是因為在thead裡面，所以用th

form：表單

< form action="register.php" method="post">

label：標籤(很好用的功能)，通常label後面會有一個屬性for，這對應到另一個標籤id，不用for的時候也可以讓使用者在點選字樣的時候，可以自動對應到目標內

    < label for="email">Email< /label>
    < input type="text" id="email">

action：行動，form裡面的屬性，把表單內容整理起來送到哪裡去處理，例如："register.php"

method：方法，有兩個值get、post

get：會把資料整理完後送出去給後台，但資料「會」顯示在網址上，較不安全

post：會把資料整理完後會從後面默默的送出去給後台，資料「不會」顯示在網址上，比較安全

input：輸入

< input type="text" placeholder="Email" required>

input：輸入：有多個type屬性，text、password、radio、checkbox

text：文字輸入

password：會將輸入內容特地隱藏(mask)起來

placeholder：佔位符號，預先顯示的內容，在input的框內預先顯示給使用者看的，方便說明此格是要輸入什麼內容，如email、姓名等

required：需要，必須填寫，html:5 在form增加的屬性

< input type="radio" name="gender" value="male">

< input type="radio" name="gender" value="female checked">

radio：單選題功能，只能選一個，另一個會消失，如性別選擇是男生或女生

    ※ name：在input屬性中的name相同會被歸類在同一組，才能區分哪些選項是單選

< input type="checkbox">

checked：檢查，預設值的意思，讓表單在一開始就先選好的選項

checkbox：多選題功能，可以同時點選很多個，如興趣有很多個

select：下拉式選項，可以定義很多種選項內容

option：選項

selected：預先顯示的，也是預設值的意思，不使用的話預設會顯示第一個

    < select name="year">
        < option value="1901">1901< /option>
        < option value="1902">1902< /option>
        < option value="1903" selected>1903< /option>
        < option value="1904">1904< /option>
        < option value="1905">1905< /option>
    < /select>

textarea：很大的文字方塊，可以留言給管理者或像聊天框的地方，可設定屬性有cols(幾行)、rows(幾列)，可以設定框的大小

< textarea cols="30" rows="10">

button：按鈕，把表單內容送出去

reset：把表單內的全部資料清掉

type="button"：只是要顯示起來是個button，但實際上沒有任何功能，常配合JavaScript去處理動作(客製化動作)

      < form action="register.php">
        < input type="text">
        < button>Submit< /button>                    <---都沒寫type預設就是submit，把東西送出去
        < button type="reset">Reset< /button>
        < button type="button">No action< /button>
      < /form>
